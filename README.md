# Tasky
Tasky app
Tasky is a cross-platform task management (To-Do) application built with **Flutter**. It allows users to add new tasks, categorize them by priority, track completed tasks, and manage their user profile with a clean and intuitive interface.

---

## Features

- Add and manage tasks easily
- View current and completed tasks
- High priority task categorization
- User profile management
- Responsive UI for Android, iOS, Web, and Desktop
- **Dark & Light themes**
- Clean and maintainable code following **Clean Code** principles

---

## Technologies & Packages Used

- **Flutter** & **Dart** — for building the app
- **Provider** — state management
- **Shared Preferences** — for storing user data and app settings locally
- **Clean Code principles** — project structured for readability and maintainability
